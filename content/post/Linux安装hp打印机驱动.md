### 一、安装hplip

`# dnf install hplip`

或者官网下载

安装hplip-gui,

`# dnf install hplip-gui`

### 二、运行hp-plugin安装插件

或者下载hplip-3.17.10-plugin.run文件，版本和hplip相同，网址为：http://www.openprinting.org/download/printdriver/auxfiles/HP/plugins/

安装时会验证文件，不用理会，直接安装就行。

好了！
