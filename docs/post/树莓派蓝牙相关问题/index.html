By wiseai | 2022-06-15 | 1 minutes
蓝牙 无线

因系统更新快，这里的操作仅供参考!!

  1. 升级安装蓝牙相关软件包

# sudo apt-get install pi-bluetooth bluez bluez-firmware blueman

  1. 添加pi用户到蓝牙组

# sudo usermod -G bluetooth -a pi

  1. 重启

# sudo reboot

  1. 使用命令打开蓝牙
pi@raspberrypi:/ $ sudo bluetoothctl
